# ZNF808 Analysis

This repository contains the scripts used for analysing the single cell datasets (scRNA and scATAC) to understand the role of ZNF808 in human pancreatic development

## Packages Used:
Seurat, Signac

### Background:
ZNF808 is a primate specific KRAB Zinc finger transcription factor, which plays a vital role during human pancreatic development. It functions by epigenetically repressing a specific class of transposable elements called the MER11 elements.
Loss of function variants in ZNF808 has been shown to cause pancreatic agenesis, a rare human developmental disorder. We aim to understand how ZNF808 controls the different layers of regulation such as chromatin accessibility, DNA methylation, and histone modifications during the development of pancreas.
